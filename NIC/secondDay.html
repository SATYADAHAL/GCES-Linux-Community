2.1. 0. whoami

Example:print effective user name

2.2. 1. ls - List Directory Contents:

Example: ls or ls -l (to show detailed information)

2.3. 2. pwd - Print Working Directory:

Example: pwd

2.4. 3. cd - Change Directory:

Example: cd /path/to/directory

2.4.1. Absolute vs relative Path

  1. Relative Path

    Relative paths are specified from the current working directory. Example: cd ../../path/to/directory

  2. Absolute Path

    Absolute paths provide the full file or directory location from the root directory. Example: cd /home/usrname/Desktop

2.5. 4. cp - Copy Files or Directories:

Example: cp sourcefile destination

2.6. 5. mv - Move (Rename) Files or Directories:

Example: mv oldname newname or mv source destination

2.7. 6. rm - Remove Files or Directories:

Example: rm filename or rm -r directory (be cautious with the -r option)

2.8. 7. shred - overwrite a file to hide its contents, and optionally delete it

Example: shred -n 10 filename

2.9. 7. mkdir - Create a Directory:

Example: mkdir -p ~/Desktop/Test/newFolder

2.10. 8. rmdir - Remove an Empty Directory:

Example: rmdir emptydirectory

2.11. 9. echo - Display a Message or Output:

Example: echo "Hello, World!"

2.12. 10. cat - Concatenate and Display File Content:

Example: cat filename

2.13. 11. man/whatis - Display Manual Pages for Commands:

Example: man commandName

2.14. 12. grep - Search Text Patterns:

Example: grep pattern filename

2.15. 13. kill - Terminate a Process:

Example: kill processId

2.16. 14. df - Display Disk Space Usage:

Example: df -h (to show human-readable sizes)

2.17. 15. du - Display File and Directory Space Usage:

Example: du -h (to show human-readable sizes)

2.18. 16. tar - Archive and Extract Files:

Example: tar -cvf archive.tar file1 file2 file3 Example: tar -xvf archive.tar –one-top-level

2.19. 17. zip - Compress or Decompress Files:

Example: gzip filename (compress) or gzip -d filename.gz (decompress)

2.20. 18. wc - print newline, word, and byte counts for each file

Example: wc filename

2.21. 19. ln - make links between files

Example: ln -s file1 sLink

2.22. 20. realpath - print the resolved path

Example:realpath filename –relative-to=[DIR]

2.23. 21. uname - print system information

Example: uname -a
