2. File Permissions

2.1. 0. getfacl

Get file access control lists Example: getfacl filename

2.2. 1. chmod

Change the permissions of a file or directory Example: chmod 644 filename.txt Example: chmod +x a.out

2.3. 2. chown

Change file owner and group of file and folder Example: sudo chown newuser:newgroup myfile.txt #Change username and group Example: sudo chown -R newuser:newgroup mydirectory #For directory Example: sudo chown :newgroup myfile.txt #Change only the group of a file

3. User Managment Command

3.1. 0. adduser

Add a new user account. This command is overlay over ~useradd~ command. Example: sudo adduser username

3.2. 1. passwd

Change the user password. Example: sudo passwd username

3.3. 2. userdel

Delete a user account. Example: sudo userdel -r username

3.4. 3. usermod

Modify user account settings. Example: sudo usermod -options username Example: sudo usermod -aG sudo username

3.5. 4. su

Switch to another user account. Example: su - username

3.6. 5. id

Display user and group information for a user. Example: id

3.7. 6. whoami

print effective user name Example: whoami

4. Group Managment Command

4.1. 0. groups

Display groups a user is a member of. Example: groups username

4.2. 1. groupadd

Add a new group. Example: sudo groupadd groupname

4.3. 2. groupdel

Delete a group. Example: sudo groupdel groupname

4.4. 3. groupmod

Modify group settings. Example: sudo groupmod -options groupname Example: sudo groupmod -n newName oldName #Rename Group

4.5. 4. gpasswd

Add or Remove user from group Example: sudo gpasswd -a username groupname # Add user to a group Example: sudo gpasswd -d username groupname # Remove user from a group

4.6. 5. newgrp

Login to new group. Example: newgrp groupname

6. Process Managment.

6.1. top

Real-time system monitoring tool that shows a dynamic view of system processes Example: top

6.2. kill

Sends a signal to terminate a process with the specified Process ID (PID). Example: kill -9 <pid>

6.3. htop

An advanced, interactive version of top that provides a more user-friendly interface for process management. Example: htop

6.4. fg and bg

Puts a process in the background. Brings a background process to the foreground. Example: ./a.out & Example: fg

6.5. ps

Displays a snapshot of currently running processes and other processes with different flags. Exampe: ps aux
